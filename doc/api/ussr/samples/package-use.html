ussr.samples.atron Simulation examples based on the ATRON modular self-reconfigurable robot. 
ussr.samples.atron.natives ATRON examples based on native (implemented in C) controllers

ussr.samples.atron.simulations Concrete examples of ATRON simulations.

ussr.samples.odin.simulations Concrete Odin simulation examples. 
ussr.samples.stickybot Small example of a spherical robot with magnetic connectors on its surface.
