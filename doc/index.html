USSR Manual

Welcome to the Unified Simulator for Self-reconfigurable Robots (USSR). This user manual provides general information on installing and using USSR.

Sections of the manual

  1. Installation: Installation instructions for installing USSR from a ZIP archive. Information on how to install USSR directly from the MMMI SVN repository is available in the file INSTALL-MMMI.txt
  2. Usage: Usage instructions for creating and running simulations in USSR
  3. Howto: How to instructions for using USSR as a framework to create new simulated robots
  4. FAQ: FAQ covering common issues in the other sections of the manual
  5. Bugs: Known bugs in the simlator (with workarounds where possible)
  6. API: JavaDoc API documentation for USSR

Getting started

Once you have completed the installation, read the usage section to learn how to run a simulation. We recommend starting with the ATRON simulations since they are more mature and the API is better documented. The next step is creating e.g. ATRON simulations of you own, also described in the usage section. For creating your own robot, follow the instructions in the howto section of the manual.
