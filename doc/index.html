USSR Manual

Welcome to the Unified Simulator for Self-reconfigurable Robots (USSR). This user manual provides general information on installing and using USSR. Note that information on how to install USSR directly from the MMMI SVN repository is available in the file INSTALL-MMMI.txt (in the root of the project).

Sections of the manual

  1. Installation: installation instructions for installing USSR from a ZIP archive.
  2. Usage: Usage instructions for creating and running simulations in USSR
  3. Howto: How to instructions for using USSR as a framework to create new simulated robots
  4. FAQ: FAQ covering common issues in the other sections of the manual
  5. API: JavaDoc API documentation for USSR
