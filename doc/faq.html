This section of the manual contains various frequently asked questions regarding use of the simulator.

Installation FAQs

Q1: Why does eclipse display an error that external libraries (jar files) cannot be found, after having checked out USSR from the MMMI SVN repository?

A1: There appears to be some cross-platform issue with the path setup. Remove all the jar files, wait for the system to build (with lots of errors), then add the jar files again.
