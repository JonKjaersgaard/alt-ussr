Package contains the major classes in relation to interactive construction of modular robot morphology, assignment of labels, controllers and saving-loading of simulation (also robot) in XML file.

This implementation is so called MODEL in MVC pattern and is responsible for covering functionality from tabs like SimulationTab(partially), ConstructRobotTab,AssignLabelsTab and AssignControllersTab.

1) How to...

1.1) Support new modular robot in ConstructRobotTab?

Look here: "ussr.builder.constructionTools"->package.html

1.2) Support new controller in AssignControllerTab?

Look here: "ussr.builder.controllerAdjustmentTool"->package.html

1.3) Support new entity for labeling in AssignLabelsTan?

Look here: ussr.builder.labelingTools --> package.html
