A physics simulation implementation based on JME (JMonkeyEngine). Implements the interfaces defined in the {@link ussr.physics} package. Provides a collection of actuators and sensors as well as various utility classes. The components for building robots are defined in the following subpackages:
  • {@link ussr.physics.jme.actuators} Actuators for connecting module components of a single module
  • {@link ussr.physics.jme.connectors} Connectors for connecting different modules
  • {@link ussr.physics.jme.sensors} Sensors for mounting on module components
Concrete robot factories are defined in the {@link ussr.physics.jme.robots} package.
