A physics simulation implementation based on JME (JMonkeyEngine). Implements the interfaces defined in the ussr.physics package. Provides a collection of actuators and sensors as well as various utility classes. The components for building robots are defined in the ussr.physics.jme.actuators, ussr.physics.jme.connectors, and ussr.physics.jme.sensors packages, whereas concrete robots are defined in the ussr.physics.jme.robots package.
