This folder is dedicated to keep different resources used by GUI.

Here:
  1. "icons" folder contains the icons of GUI in .png format.
    • Most of icons are 20x20 pixels
    • Big (keyword in the name of the file) are 25x25 pixels
    • Small (keyword in the name of the file) are 15x15 pixels
  2. Icons were created in icon editor called RealWorldIconEditor.
  3. Note: Each icon as well as the directory it is in, are referenced from Interfaces and Enumerations in the code for GUI
  4. Raw design of each icon can be found in folder named as rawIcons. This is in case, there is a need to modify the design.
