This folder is dedicated to keep different resources used by GUI.

Here:
1)"icons" folder contains the icons of GUI in jpg format.
Dimensions of icons:
A) in toolBars - 20x20 pixels.
B) in menu bar - 15x15 pixels.
Hint: Each icon as well as the directory it is in, is referenced from "MainFrameInter" interface located in package "ussr.aGui"
2)"MSVisio" folder contains the files for improving the design of icons and saving in jpg format.
